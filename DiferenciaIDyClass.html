Parte de invesigación

Las principales diferencias entre class e id en CSS són las siguientes:

1. Un identificador (id) solo se puede usar una vez por página un unico elemento, mientras que class se puede usar en multiples elementos en una misma página.

2. Los identificadores tienen mayor prioridad en el CSS respecto a class, por lo que si hay un elemento con ambos, solo cargara el de la id por prioridad.

3. Se recomienda usar el id para los elementos únicos, como por ejemplo para un formulario en específico. Mientras que las clases sirven para que distintos elementos tengan el mismo estilo.
